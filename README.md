# Trilha JS Developer - Pokedex

Desafio de projeto da DIO. Peguei o projeto base e fui mexendo pra deixar com a minha cara.

## o que eu fiz

- montei o consumo da PokeAPI no arquivo `pokeApi.js`
- listagem com paginação (botão de carregar mais)
- busca por nome ou número
- filtro por tipo (puxa os tipos da própria api)
- modal com os detalhes do pokemon quando clica no card (stats, altura, peso, habilidade)
- redesenhei o css inteiro pra ficar com cara de pokedex mesmo

## como rodar

só abrir o index.html no navegador. precisa de internet por causa da api.

## tecnologias

html, css e javascript puro. sem framework nenhum.
